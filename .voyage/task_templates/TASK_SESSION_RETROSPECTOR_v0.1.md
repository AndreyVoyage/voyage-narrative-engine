## Goal
Реализовать минимальный `session_retrospector.py`, который:
- принимает raw session log (`.md` или `.log`);
- делит на кадры (frames) с таймстампами;
- определяет speaker (User / NPC / System);
- создаёт `RETRO_FRAMES.jsonl` — машиночитаемый аудит;
- создаёт `RETRO_SUMMARY.md` — человекочитаемый отчёт.

## Acceptance Criteria
- [ ] `session_retrospector.py` запускается с `--help` и показывает справку.
- [ ] Принимает путь к raw `.md` или `.log` через аргумент `--input`.
- [ ] Создаёт список кадров (frames) с таймстампами или порядковыми номерами.
- [ ] Определяет `speaker` (User / NPC / System) для каждого кадра.
- [ ] Сохраняет `RETRO_FRAMES.jsonl` с полями: `timestamp`, `speaker`, `type`, `content`.
- [ ] Сохраняет `RETRO_SUMMARY.md` с общей статистикой, ошибками и рекомендациями.
- [ ] Не редактирует `personas/`.
- [ ] Не редактирует `core/`.
- [ ] Не требует LLM API — работает offline (regex + JSONL).
- [ ] Код проходит `ruff check` (если применимо).
