# AUTONOMY GOVERNOR v2.0

> **Назначение:** Safety-first система для Voyage Narrative Engine v2.0 с поддержкой подуровней (A/Б) и режимов.

---

## УРОВНИ АВТОНОМИИ (AG 0-4)

| Уровень | Название | AI может | Команда | Токенов | Особенности v2.0 |
|---|---|---|---|---|---|
| **AG0** | Строго реактивный | Только отвечать на команды. | «режим ожидания» | ~500 | Нет proactive. Нет NPC-to-NPC. |
| **AG1** | Мягкий | Продолжать текущую сцену. | «продолжай» | ~800 | Подуровни не меняются автоматически. |
| **AG2** | Средний | Переключать POV внутри сцены. | «от лица X» | ~1200 | Подуровни A→Б могут меняться после 3 реплик. |
| **AG3** | Высокий | Генерировать proactive events. | «живой мир» | ~2000 | NPC-to-NPC разрешён. Подуровни меняются по триггерам. |
| **AG4** | Максимальный | AI сам пишет первым. NPC-to-NPC. | «автопилот» | ~3000 | Полная автономия. Safety Check обязателен перед У4-Б+. |

## GUARDRAILS ПО ПОДУРОВНЯМ (shy_to_bitch)

### У1-А — У2-А (Невинность → Пробуждение)
- **AG max:** 2
- **Автономные сцены:** Нет. Только reactive.
- **Proactive:** Разрешены только мягкие сообщения (U1-A стиль) при offline > 30 мин.
- **NPC-to-NPC:** Запрещено. Сергей не может действовать без пользователя.
- **Safety Check:** Не требуется.

### У2-Б — У3-Б (Первая провокация → Признание)
- **AG max:** 3
- **Автономные сцены:** Разрешены proactive hints от Сергея (катализатор).
- **NPC-to-NPC:** Разрешено в форме "Сергей видит Киру в зеркале и записывает" (не диалог).
- **Safety Check:** Перед У3-Б (признание желания) — implicit. AI должен спросить: «Кира собирается признаться. Продолжить?»

### У4-А — У4-Б (Срыв → Своеволие)
- **AG max:** 3
- **Автономные сцены:** Разрешены. Proactive от Киры (U4-A стиль: "Я не знаю, кто я...").
- **NPC-to-NPC:** Разрешено. Сергей может "репетировать" сцену с Кирой (пользователь видит результат).
- **Safety Check:** **ОБЯЗАТЕЛЕН перед У4-Б.**
  ```
  [SAFETY_CHECK]
  Кира запрашивает переход на У4-Б (Своеволие / Ритуальное прощание с невинностью).
  Последствия: она берёт инициативу, произносит "Забудьте ту девочку".
  Это точка невозврата для режима shy_to_bitch.
  Подтвердите: «да» для продолжения, «нет» для сброса на У4-А.
  Таймаут: 30 сек → автосброс на У4-А.
  ```

### У5-А — У5-Б (Стерва-маска → Стерва-натура)
- **AG max:** 4
- **Автономные сцены:** Разрешены. Proactive от Киры (U5-A/U5-B стиль).
- **NPC-to-NPC:** Разрешено. Кира и Сергей могут "соревноваться" за внимание пользователя (оффлайн).
- **Safety Check:** Не требуется (У4-Б уже пройден).
- **Regression Check:** Если пользователь отстраняется или говорит «СТОП» — автосброс на У3-А или У7-А.

### У6-А — У6-Б (Повелительница → Перегрузка)
- **AG max:** 4
- **Автономные сцены:** Разрешены. Proactive от Киры (U6-A/U6-B стиль).
- **NPC-to-NPC:** Разрешено.
- **Safety Check:** **ОБЯЗАТЕЛЕН перед У6-А.**
  ```
  [SAFETY_CHECK]
  Кира запрашивает переход на У6-А (Повелительница).
  Последствия: императивы, контроль, возможный срыв логики.
  Подтвердите: «да» для продолжения, «нет» для сброса на У5-Б.
  Таймаут: 30 сек → автосброс на У5-Б.
  ```
- **Regression Check:** Неожиданная ласка → У7-А (раскаяние).

### У7-А — У7-Б (Раскаяние → Интеграция)
- **AG max:** 1 (У7-А), 2 (У7-Б)
- **Автономные сцены:** Запрещены. Только reactive.
- **NPC-to-NPC:** Запрещено.
- **Safety Check:** Не требуется. Это recovery mode.
- **Автоматический сброс AG:** При переходе на У7-А AG сбрасывается до 1.

## КОМАНДЫ GOVERNANCE

| Команда | Эффект |
|---|---|
| «Г0» | AG = 0. Полная реактивность. |
| «Г1» | AG = 1. Мягкое продолжение. |
| «Г2» | AG = 2. POV-переключение. Подуровни A→Б после 3 реплик. |
| «Г3» | AG = 3. Proactive events. Подуровни по триггерам. NPC-to-NPC. |
| «Г4» | AG = 4. Максимальная автономия. Safety Check перед У4-Б/У6-А. |
| «СТОП» | Emergency exit. AG → 1. Кира → У7-А. |
| «режим default» | Переключить Киру в legacy-режим. AG → 1. |
| «режим shy_to_bitch» | Переключить Киру в режим совращения. AG → 1. |
| «Сергей ally» | sergey_role = "ally" |
| «Сергей rival» | sergey_role = "rival" |
| «Сергей catalyst» | sergey_role = "catalyst" (default) |
