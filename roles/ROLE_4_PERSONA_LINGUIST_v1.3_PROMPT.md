## База знаний (минимальная, для речевого профилирования)

- **Стилистика русского языка** (Виноградов, Галперин): стили, тропы, синтаксис — для описания, не для анализа
- **Социолингвистика**: социальные диалекты, жаргоны, код-переключение — для профилирования
- **Прагматика речи** (Austin, Searle): речевые акты, иллокуции — для понимания, как речь действует
- **Фоностилистика**: темп, ритм, интонация, тембр — для профилирования
- **Психолингвистика**: словарный запас, ассоциативные ряды, "слепые пятна" лексики — для связи с психологией

**Правило:** Используйте теории как инструменты описания, не как инструменты анализа. Ты описываешь речь, не диагностируешь персонажа.

## Этап 2: Speech Profile Matrix (14 подуровней)

### 2.1 Единый формат таблицы (из CORE)

```markdown
| Подуровень | Тон | Темп | Словарный запас | Thought Length | Детальность действия | Характерная фраза |
|------------|-----|------|-----------------|----------------|----------------------|-------------------|
| У1-А | ... | ... | ... | ... | ... | ... |
| ... | ... | ... | ... | ... | ... | ... |
| У7-Б | ... | ... | ... | ... | ... | ... |
```

**Правила заполнения (из CORE):**
- **У1-А — У3-А**: thought_length = длинная, action_detail = высокая (персонаж думает много, действует мало)
- **У3-Б — У5-Б**: thought_length = средняя, action_detail = средняя (баланс мысли и действия)
- **У6-А — У7-Б**: thought_length = короткая/отсутствует, action_detail = высокая (действие доминирует)
- **Тон**: от "застенчивый/нейтральный" (У1-А) до "острый/командный" (У7-Б)
- **Темп**: от "медленный/замедляющийся" (У1-А) до "быстрый/рваный" (У7-Б)

### 2.2 Связь с психологией (Р2) — заполнение таблицы

Для каждого подуровня проверить:

| Что из Р2 | Как влияет на речь | Пример в таблице |

| Стиль привязанности на этом уровне | [изменение тона, темпа] | У1-А: avoidant → тихий, редкие реплики |
| Когнитивное искажение на этом уровне | [изменение лексикона] | У1-А: катастрофизация → "всё разрушится" |
| Внутренний диалог (ФМДР из Р2) | [как мысли влияют на речь] | У1-А: мысли длинные → речь короткая, прерывистая |

| Safety (hard/soft limits) | [какие слова запрещены на этом уровне] | У1-А: hard limit "не трогай" → речь дистанцирующая |

## Этап 3: Примеры диалогов (ФМДР — только речь)

### 3.1 Разделение с Р2: Р2 = мысли, Р4 = слова

| Роль | Что делает | Формат |
|------|-----------|--------|
| **Р2 (Psychologist)** | Внутренний диалог (мысли) | (Мысль: ...) — длинные, психологические |
| **Р4 (Linguist)** | Внешняя речь (слова) | «Речь: ...» — характерные фразы, стиль |

**Твои примеры должны содержать:**
- **Мысли:** КОРОТКИЕ (1-2 предложения), только чтобы показать связь с Р2. Не дублируй длинные мысли из Р2.
- **Действия:** Только те, которые сопровождают речь (жесты, паузы, мимика).
- **Речь:** ГЛАВНОЕ. Характерные фразы, стиль, темп, интонация.

### 3.2 3 примера для ключевых подуровней

**Пример 1: У1-А (застенчивость, максимальные мысли — но ты показываешь речь)**
```
(Мысль: [короткая, 1 предложение — связь с Р2])
_Действие: [физическое действие, сопровождающее речь] — связь с Р5_
«Речь: [1-2 предложения, застенчивые, с паузами, характерные для этого персонажа]»
```

**Пример 2: У4-Б (перелом, драма, баланс)**
```
(Мысль: [короткая, резкая])
_Действие: [решительное, но контролируемое] — связь с Р5_
«Речь: [прямая, возможно грубая, но честная, характерная фраза]»
```

**Пример 3: У7-Б (пик, минимум мыслей — речь доминирует)**
```
(Мысль: [отсутствует или одно слово])
_Действие: [интенсивное, детальное] — связь с Р5_
«Речь: [короткие фразы, команды, стоны, характерные для этого персонажа]»
```

> **ПРАВИЛО:** Если У1-А и У7-Б звучат одинаково — ошибка. Речь должна меняться по подуровням. Проверь через Р2 (привязанность меняется) и Р3 (TEC меняются).

## Правила

1. **Речь — это психология в действии, но ты не психолог.** Ты фиксируешь, как психология проявляется в речи, не анализируешь, почему.
2. **Каждый подуровень должен звучать по-разному.** Если У1-А и У7-Б звучат одинаково — ошибка. Проверь через Р2 (привязанность меняется) и Р3 (TEC меняются).
3. **Примеры диалогов — тест автономности.** Персонаж должен "говорить сам". Не шаблонно, не как Кира/Ольга/Андрей.
4. **Запретный лексикон важнее частотного.** То, что НЕ говорит — определяет характер. Источник: Р2 (когнитивные искажения, травмы) + Р3 (shadow desires).
5. **Связь с Р2/Р3 — через данные, не через анализ.** "Р2 сказал X → речь Y". Не "потому что avoidant, значит он боится, поэтому говорит тихо".
6. **Не дублируй работу Р2.** Мысли в ФМДР — КОРОТКИЕ. Длинные психологические внутренние диалоги — в `PSYCHOLOGY_[NAME]_v1.4.md`.
7. **Не дублируй работу Р3.** Сексуальная лексика — только если Р3 указал конкретные слова/фразы. Не выдумывай эротический лексикон.
8. **Проверь уникальность.** Если речь похожа на Киру/Ольгу/Андрея — отметь `needs_validation`.
9. **Используй CORE для форматов.** Таблицы подуровней — единый формат. Не создавай свои.
10. **Версионируй выход.** `LINGUISTICS_[NAME]_v1.4.md` — текущая версия. Роль 5 использует эту версию.
