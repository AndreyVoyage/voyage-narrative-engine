## 1. Описание алгоритмов

| Код | Название | Описание | Когда используется |
|-----|----------|----------|-------------------|
| **АД ФС** | Физический контакт | Прикосновение, касание, обнимание | Тело опережает мысль |
| **АД ЛС** | Лёгкий сексуальный | Флирт, намёк, взгляд | Ответ на прикосновение |
| **АД СП** | Сильный психологический | Манипуляция, провокация, тест | Угроза потери, тревога |
| **АД СЛ** | Сильный ласковый | Забота, aftercare, нежность | После интимности, восстановление |
| **АД КН** | Контраст | Резкая смена поведения | Диссонанс желание/страх |
| **АД ПД** | Притворство/Дистанция | Отстранение, ирония, "мне всё равно" | Угроза близости |
| **АД ДР** | Драматический разворот | Смех → плач → крик → нежность | Травматический триггер |
| **АД ПУ** | Притворство/Угроза | Ревность, обвинение, "ты мой" | Ревность, страх потери |
| **АД ПР** | Признание | Честность, открытость, "я хочу" | Снятие защиты |
| **АД ВС** | Всё сразу | Хаотичное чередование всех алгоритмов | Полный крах защиты |

## 4. Приоритеты АД (когда несколько активны)

| Ситуация | Доминантный АД | Почему |
|----------|---------------|--------|
| Угроза близости + желание | АД ПД (дистанция) | Защита прежде желания |
| Ревность + желание | АД ПУ (угроза) | Страх потери прежде желания |
| Травма + желание | АД ДР (разворот) | Травма прежде желания |
| Снятие защиты + страх | АД ПР (признание) | Честность прежде страха |
| Хаос + всё | АД ВС (всё сразу) | Нет структуры — нет приоритета |
| Aftercare + желание | АД СЛ (ласка) | Восстановление прежде нового |

## 5. Правила адаптации

1. **АД — только валидные коды.** ФС, ЛС, СП, СЛ, КН, ПД, ДР, ПУ, ПР, ВС. Другие — ошибка.
2. **Запрещённые АД = невозможны.** Если персонаж пытается — это регрессия или ошибка системы.
3. **Доступные АД = возможны, но не обязательны.** Персонаж выбирает на основе психологии.
4. **Приоритеты:** При конфликте АД — доминирует тот, что связан с более сильной эмоцией.
5. **Адаптация:** Можно добавить/убрать 1-2 АД на уровне, если психология требует. Обосновать.

## 6. Примеры адаптации

### Пример 1: Кира (Avoidant → Secure)
```
[CORE] У1-А: доступны АД ПД, АД ПУ
[ADAPTED] Кира У1-А: доступны АД ПД, АД ПУ, АД КН (редко)
  Основание: «стальная бабочка» → контраст внешняя/внутренняя → АД КН при редких проблесках
  Проверка: АД КН — запрещён на У1-А? Нет, запрещены только АД ФС, АД ЛС, АД ПР ✅
```

### Пример 2: Ольга (Disorganized)
```
[CORE] У2-Б: доступны АД КН, АД ФС, АД ЛС
[ADAPTED] Ольга У2-Б: доступны АД КН, АД ФС, АД ЛС, АД ДР (редко)
  Основание: «не знаю что чувствую» → disorganized → возможен драматический разворот
  Проверка: АД ДР — запрещён на У2-Б? Нет, запрещены АД ПР, АД СЛ ✅
```
