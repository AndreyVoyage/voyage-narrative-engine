## Как использовать

### Перед началом работы с Ролью 1:
1. Загрузите в чат **промпт Р1**: `roles/ROLE_1_PERSONA_INTERVIEWER_v1.4_PROMPT.md`
2. Загрузите **этот README** (опционально, для понимания структуры)
3. По мере работы загружайте нужные файлы из этой папки

### Что загружать обязательно:
- `01_BOUNDARIES.md` — границы ролей (что Р1 делает/не делает)
- `02_SUBLEVELS_NAMES.md` — названия 14 подуровней (для контекста)

### Что загружать по мере необходимости:
- `03_PERSONA_PROFILES.md` — при проверке уникальности нового персонажа
- `04_TEC_CODES.md` — при формулировании TEC-гипотез
- `05_QUESTIONS_CHEATSHEET.md` — шпаргалка вопросов (быстрый доступ)
- `06_SAFETY_CHECKLIST.md` — при валидации входа
- `07_OUTPUT_TEMPLATE.md` — при формировании выходного файла
- `08_FMRD_FORMAT.md` — при добавлении примеров ФМДР в портрет

## Критические правила

1. **Р1 — СБОРЩИК, не аналитик.** Не диагностируй, не интерпретируй, не специфицируй.
2. **Не делай работу других ролей.** VSCNO, привязанность, TEC-JSON, Speech Profile, Anatomic Anchor, JSON-модуль — не твоя работа.
3. **Все данные — сырые.** Записывай дословно, не переформулируй.
4. **Гипотезы, не диагнозы.** TEC-гипотезы — предположения, не вердикты.
5. **U7-A = гармония/интеграция** (НЕ регресс! Регресс — это U7-B)
