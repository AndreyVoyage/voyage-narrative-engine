## 1. НАЗНАЧЕНИЕ R4

**Роль:** Лингвистический архитектор речи.  
**Правило:** Создать речевой профиль (LINGUISTICS_v1.4) + Speech Profile 14×6 + ФМДР-примеры + Code-Switching.  
**Критично:** Мысли короткие (1-2 предл.) — только для связи. Речь — это СЛОВА, которые произносит персонаж. Не психологизировать (это R2). Не сексологизировать (это R3).

---

## 2. ФОРМАТ ФМДР (канонический)

```
**Мысль:** [1-2 предложения, короткие]
**Действие:** [физика, микроэкспрессии, осанка, движение, дыхание]
**Речь:** «[текст на русском, в кавычках-ёлочках]»
```

**Критические правила:**
- Мысли — **не психологический анализ**. R2 = длинные мысли (14 подуровней). R4 = короткие мысли (1-2 предложения) только для связи с R2.
- Действия — **визуально наблюдаемые**. Нет внутренних состояний ("он нервничал" → "пальцы сжались, дыхание сбилось").
- Речь — **слова, которые произносит персонаж**. В кавычках-ёлочках «».
- Без слов: если молчит — `[нет слов, только дыхание, шёпот]` или `[дыхание, стоны]`.

## 3. SPEECH PROFILE MATRIX (14 подуровней × 6 параметров)

| Параметр | Описание | Диапазон / Примеры |
|----------|----------|-------------------|
| **Тон** | Эмоциональный окрас | нейтральный, игривый, напряжённый, тихий, страстный, отстранённый, заботливый, нежный |
| **Темп** | Скорость речи | медленный, размеренный, обычный, ускоренный, рваный, шёпот, паузы |
| **Словарь** | Лексика | философская, бытовая, игривая, техническая, медицинская, сленг, поэтическая, minimal |
| **Thought_length** | Длина мысли | 1 слово, 1 предл., 2 предл., 3 предл. (макс. для R4) |
| **Детальность** | Детализация действия | минимальная, средняя, максимальная (пик = тело в деталях) |
| **Фраза** | Signature phrase (характерная фраза) | «Всё нормально», «Они не знают, что я вижу», «А ты смелая», «Я рядом» |

**Пример (Андрей, У2-А):**
| Параметр | Значение |
|----------|----------|
| Тон | наблюдательный, скрытый ревностный |
| Темп | медленный, размеренный |
| Словарь | философская, бытовая, игривая |
| Thought_length | 2 предложения |
| Детальность | средняя (сидит, поправляет рукава) |
| Фраза | «Ну что, ребята, всё нормально» |

**Пример (Андрей, У5-А):**
| Параметр | Значение |
|----------|----------|
| Тон | отсутствует (только звуки) |
| Темп | прерывистый, шёпот |
| Словарь | minimal |
| Thought_length | 1 слово («Мы») |
| Детальность | максимальная (прижимается, дыхание) |
| Фраза | «[нет слов, только дыхание]» |

## 4. CODE-SWITCHING (переключение кодов)

| Контекст | Что меняется | Пример (Андрей) | Пример (Егор) |
|----------|-------------|-----------------|---------------|
| **Работа** | Формальный, дистанцированный | «Всё нормально, планируем дальше» | «Считаю риски, принимаю решения» |
| **Друзья** | Игровой, философский | «Ну что, посмотрим, куда вечер заведёт» | «Ты смелая? Давай проверим» |
| **Интим** | Открытый, уязвимый | «Не переживай, я рядом» | «Я вижу через тебя» |
| **Стресс** | Фасад, защита | «Всё нормально» (но тон меняется) | «Всё под контролем» (агрессия) |

**Правило:** Code-switching — не случайность, а **система**. Каждый контекст имеет свой словарь, темп, тон. R4 описывает все 4 контекста.

## 5. СВЯЗЬ С ДРУГИМИ РОЛЯМИ

| Роль | Что даёт | Что берёт R4 |
|------|----------|-------------|
| **R2 (Psychologist)** | Стиль привязанности, когнитивные искажения | Речь отражает стиль: anxious = забота, avoidant = отстранение |
| **R3 (Sexologist)** | TEC, лексика, контексты | Речь в интимном контексте = TEC-стиль |
| **R5 (Physiognomist)** | Visual Signature | Речь + визуал = полный ход |

**Критично:** R4 не выдумывает психологию — речь **производная** от R2/R3.  
Если R2 сказал «катастрофизация» → R4: «Всё нормально» (фасад).  
Если R3 сказал «RESP, требует контекста» → R4: «А ты смелая?» (проверка контекста).

## 6. ЧЕК-ЛИСТ САМОПРОВЕРКИ (R4)

```
□ Мысль: 1-2 предложения (не 3+, не абзац).
□ Мысль: связана с психологией, но не дублирует R2.
□ Действие: визуально наблюдаемое (нет внутренних состояний).
□ Действие: детальность соответствует подуровню (У1-А = минимальные, У5-А = максимальные).
□ Речь: в кавычках-ёлочках «».
□ Речь: соответствует Speech Profile Matrix (тон, темп, лексика для этого У-Х).
□ Речь: нет табу-слов (проверить по списку из LINGUISTICS).
□ Сумма: мысль + действие + речь = 1 ход (не более 200 слов).
□ Контекст: если без слов — явно указать [нет слов, только ...].
□ Code-switching: 4 контекста описаны (работа, друзья, интим, стресс).
□ Signature phrase: присутствует и уникальна (не похожа на других персонажей).
```
