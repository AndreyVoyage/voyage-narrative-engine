# 🔧 PROMPT ASSEMBLY GUIDE — Как собрать сессию из модулей

> **Цель:** За 3 шага собрать полный промпт для нейросети из ядра + персонажей + состояния + сценария.
> **Результат:** PROMPT.txt готов к копированию в чат.

---

## 📋 ШАГ 1: Подготовка файлов

### Необходимые файлы:

```
1. VOYAGE_NARRATIVE_CORE.md      — ядро (не менять)
2. KIRA_MODULE.json              — модуль Киры (можно редактировать переменные)
3. SERGEY_MODULE.json            — модуль Сергея (можно редактировать переменные)
4. STATE.json                    — состояние текущей сессии (создаём новый или обновляем)
5. SCENARIO.json                 — сценарий (выбираем из матрицы или создаём)
```

### Где взять:

- **CORE** — всегда один и тот же файл. Копируем целиком.
- **PERSONAS** — JSON-модули. При необходимости меняем переменные (уровень, одежда, локацию).
- **STATE** — берём STATE_TEMPLATE.json, переименовываем в STATE.json, обновляем текущие значения.
- **SCENARIO** — берём из SCENARIO_MATRIX.json или создаём вручную.

---

## 📋 ШАГ 2: Сборка PROMPT.txt

### Формула сборки:

```
PROMPT.txt = [CORE текст] + [KIRA текст] + [SERGEY текст] + [STATE текст] + [SCENARIO текст]
```

### Правила сборки:

1. **CORE** — копируем весь текстовый файл. Это ~2000 токенов.
2. **KIRA** — JSON преобразуем в текст:
   ```
   Ты — Кира. 24 года. Светло-русые вьющиеся волосы чуть ниже плеч. Зелёные глаза.
   Естественная грудь больше 3-го размера, розовые ореолы и соски.
   Текущий уровень: У1. Текущая грань: ТГ1. Локация: дом.
   Тайное желание: [из psychology.secret_desire].
   Травматический якорь: [из psychology.trauma_anchor].
   ```
3. **SERGEY** — аналогично из JSON:
   ```
   Ты — Сергей. 27 лет. Кроссфит, подтянутый. Голубые глаза, лёгкая щетина.
   Татуировка на плече. Рубашка нараспашку. Харизматичный, улыбающийся хищник.
   Текущий уровень: С1. Локация: зал.
   ```
4. **STATE** — копируем JSON как есть. Нейросеть понимает структуру.
5. **SCENARIO** — описываем сцену 2-3 предложениями или копируем JSON.

### Пример собранного начала:

```
[CORE]
```

## 📋 ШАГ 3: Запуск и управление

### Отправка в нейросеть:

1. Копируем весь PROMPT.txt в чат.
2. Ждём первое сообщение Киры (оно генерируется автоматически по ядру).
3. Отвечаем как Я (пользователь).

### Команды во время сессии:

| Команда | Когда использовать |
|---------|-------------------|
| «У2» | Ускорить переход на следующий уровень |
| «У6» | Тестовый переход на У6 (ручное управление) |
| «Г3» | Включить «живой мир» (proactive events) |
| «Г4» | Автопилот (максимальная автономность) |
| «СТОП» | Экстренный выход из любой роли |
| «Суммаризируй» | Сжать контекст при переполнении |
| «В» | Сгенерировать Qwen-промпт для текущей сцены |
| «Проверь память» | Вывести флаги и события |

### Сохранение в конце сессии:

1. Просим нейросеть: «Выведи текущий STATE.json».
2. Копируем JSON, сохраняем в файл.
3. Обновляем EVENT_LOG.json (если вели записи).
4. Git commit (если используем репозиторий).

## 🔧 АВТОМАТИЗАЦИЯ (для продвинутых)

### Bash-скрипт сборки:

```bash
#!/bin/bash
# assemble.sh

cat VOYAGE_NARRATIVE_CORE.md > PROMPT.txt
echo -e "
[ПЕРСОНАЖИ]
" >> PROMPT.txt
python3 json_to_text.py KIRA_MODULE.json >> PROMPT.txt
python3 json_to_text.py SERGEY_MODULE.json >> PROMPT.txt
echo -e "
[СОСТОЯНИЕ]
" >> PROMPT.txt
cat STATE.json >> PROMPT.txt
echo -e "
[СЦЕНАРИЙ]
" >> PROMPT.txt
cat SCENARIO.txt >> PROMPT.txt

echo "PROMPT.txt собран. Токенов: $(wc -m < PROMPT.txt)"
```

### Python-конвертер JSON → текст:

```python
# json_to_text.py
import json, sys

with open(sys.argv[1]) as f:
    data = json.load(f)

print(f"Ты — {data['name']}. {data['variables']['age']} лет.")
for k, v in data['variables'].items():
    if k != 'age':
        print(f"{k}: {v}.")
print(f"Психология: {data['psychology']}")
print(f"Алгоритмы: {', '.join(data['algorithms'])}")
```
