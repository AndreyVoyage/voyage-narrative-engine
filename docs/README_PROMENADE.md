# 📖 VOYAGE PROMENADE v2.0 — Инструкция по запуску

## Быстрый старт (3 шага)

### Шаг 1: Загрузите файлы в репозиторий
```bash
# Скопируйте файлы из архива в папку проекта:
cp scenarios/SCENARIO_PROMENADE_v2.md ~/voyage-narrative-engine/scenarios/
cp personas/FEMALE_USER_MODULE.json ~/voyage-narrative-engine/personas/
cp state/STATE_TEMPLATE_PROMENADE.json ~/voyage-narrative-engine/state/
```

### Шаг 2: Соберите промпт
```bash
cd ~/voyage-narrative-engine
bash tools/assemble.sh promenade
# Результат: full_prompt_promenade.txt
```

### Шаг 3: Запустите сессию
1. Откройте чат-бот (Kimi / DeepSeek / Qwen).
2. Вставьте содержимое `full_prompt_promenade.txt` в системный промпт или первое сообщение.
3. Напишите: `Начни` или `Старт`.
4. Система спросит: **«Как тебя зовут?»** — ответьте любым именем.
5. Начнётся сценарий с вашим именем в повествовании и диалогах.

## 🎭 Что внутри сценария

- **Пролог «Свобода»** — героиня одна на набережной, в лёгком платье, без белья. Эксперимент со свободой.
- **Акт I «Встреча»** — Максим подходит первым (нежность), Сергей остаётся на скамейке (интрига).
- **Акт II «Притяжение»** — трое идут по набережной. Контраст начинает работать.
- **Акт III «Треугольник»** — мужчины как слаженный дуэт. Нарастание напряжения.
- **Акт IV «Кульминация»** — секретное место под мостом. Выбор: уйти, остаться с одним, или с обоими.
- **Акт V «Интим»** — круг желания Бассон в действии. Нежность + огонь.
- **Эпилог «Aftercare»** — интеграция, слёзы, плед, рука в руке.
- **Романтическая арка** — меню выбора свиданий: ресторан, кино, готовка, фуд-корт, траттория, Dining in the Dark.
