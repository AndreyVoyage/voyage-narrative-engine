## 🚀 ЗАПУСК И УПРАВЛЕНИЕ СЕССИЕЙ

| Команда | Действие | Пример |
|---------|----------|--------|
| `Начни` / `Старт` / `Start` | Запуск сценария с нуля | `Начни` |
| `Продолжи` / `Дальше` | Продолжить после паузы | `Продолжи` |
| `Медленнее` / `Погоди` | Замедлить темп, больше диалога | `Медленнее` |

| `СТОП` / `ХВАТИТ` / `КРАСНАЯ КАРТОЧКА` | Мгновенный aftercare | `СТОП` |

## 🔥 МНЕМОНИКА ВСНО (управление динамикой)

Пишите сокращения — AI перестраивает тон сцены.

| Команда | Эффект | Пример |
|---------|--------|--------|
| `ВЛ0`–`ВЛ4` | Уровень влюблённости | `ВЛ3` — она влюблена, но ещё не привязана |
| `СТ0`–`СТ4` | Уровень страсти | `СТ2` — пламя, поцелуи, прикосновения |
| `НЖ0`–`НЖ4` | Уровень нежности | `НЖ4` — полное слияние, aftercare |
| `ОГ0`–`ОГ4` | Контраст мужчин | `ОГ4` — максимальная вспышка, они работают идеально |

**Комбинации:**
- `ВЛ2 СТ1 НЖ3 ОГ2` — она интересуется, но страсть низкая, нежность высокая, контраст средний.
- `ВЛ3 СТ4 НЖ2 ОГ4` — пиковая сцена, перед интимом.

## 🎭 ПЕРЕКЛЮЧЕНИЕ ТОЧКИ ЗРЕНИЯ (POV)

| Команда | Что происходит |
|---------|----------------|
| `От лица девушки` / `POV героини` | Повествование от её имени, её мысли и чувства |
| `От лица Максима` / `POV Максима` | Его внутренний монолог, он видит её и реагирует |
| `От лица Сергея` / `POV Сергея` | Его холодный анализ, макиавеллианские мысли |
| `Три POV` / `Все вместе` | Чередование: девушка → Максим → Сергей в одном ответе |

## 🍽️ ВЫБОР СЦЕНАРИЯ (Меню свиданий)

Когда мужчины предложат варианты, выберите одно словом или фразой:

| Ваш выбор | Переход | ВСНО-эффект |
|-----------|---------|-------------|
| `Ресторан` / `крыша` / `на крыше` | Сценарий "Ресторан" | ВЛ+2, СТ+1, НЖ+1, ОГ+2 |
| `Траттория` / `итальянский` / `пицца` | Сценарий "Траттория" | ВЛ+1, НЖ+3, ОГ+1 |
| `Темнота` / `Dining in the Dark` / `в темноте` | Сценарий "Темнота" | СТ+4, НЖ+2, ОГ+3 |
| `Фуд-корт` / `уличная еда` / `шаверма` | Сценарий "Фуд-корт" | ВЛ+2, НЖ+2 |
| `Готовка` / `готовим вместе` / `кухня` | Сценарий "Готовка" | ВЛ+3, СТ+2, НЖ+3, ОГ+2 |
| `Кино` / `фильм` / `квартира` | Сценарий "Кино" | ВЛ+2, СТ+3, НЖ+2, ОГ+3 |
| `И то, и другое` / `всё сразу` | Сценарий "Полный вечер" | ВЛ+3, СТ+3, НЖ+3, ОГ+4 |

## 💾 ПАМЯТЬ И КОНТЕКСТ

| Команда | Действие |
|---------|----------|
| `Запомни` / `Помни` | Фиксация важного факта в STATE |
| `Как меня зовут?` | AI напоминает имя героини |
| `Моё имя [имя]` | Сменить имя героини (только в начале сессии) |
| `Что было раньше?` / `Вспомни` | AI кратко пересказывает предыдущие акты |

## 🤖 УРОВЕНЬ АВТОНОМИИ (Г — Губернатор)

| Команда | Режим | Что делает AI |
|---------|-------|---------------|
| `Г0` | Строго реактивный | Только отвечает на ваши команды |
| `Г1` | Мягкий | Продолжает текущую сцену, но ждёт вас |
| `Г2` | Средний | Переключает POV сам, но по вашей теме |
| `Г3` | Высокий | Proactive: мужчины могут писать первыми, диалоги NPC→NPC |
| `Г4` | Максимальный | AI пишет первым, полная автономия сюжета |

## 🎬 СПЕЦИАЛЬНЫЕ КОМАНДЫ

| Команда | Эффект |
|---------|--------|
| `ФМДР` | AI объясняет, что такое формат (Мысли → Действия → Речь) |
| `ВСНО` | AI объясняет мнемонику и текущие значения |
| `Состояние` / `State` | AI показывает текущий STATE.json |
| `Флаги` / `Flags` | AI показывает, какие события уже произошли |
| `Сброс` / `Reset` | Вернуться к началу сценария (с подтверждением) |
| `Перейти к [акт]` | Прыжок к конкретному акту (например, `Перейти к Акту III`) |

## 💡 ПРИМЕРЫ СЕАНСОВ

### Пример 1: Замедление
```
Пользователь: Медленнее
AI: (ВСНО замедляется. Больше диалога, меньше действия. Максим спрашивает о её детстве. Сергей слушает, не перебивает.)
```

### Пример 2: Эскалация
```
Пользователь: ВЛ3 СТ3 ОГ3
AI: (Сцена нагревается. Сергей делает резкий комплимент. Максим нервно смеётся, но не отступает. Девушка краснеет, но не отводит взгляд.)
```

### Пример 3: Выбор сценария
```
Максим: «У нас есть пара идей...»
Пользователь: Кино
AI: (Переход к Сценарию "Кино". ВЛ+2, СТ+3, НЖ+2, ОГ+3. Мужчины ведут её в кинотеатр.)
```

## ⚠️ ОГРАНИЧЕНИЯ

- **Имя** можно задать только в начале сессии. Позже — только через `Reset`.
- **Safety Check** нельзя пропустить при переходе на СТ4. AI спросит: «Ты уверена?»
- **Г4** (максимальная автономия) доступен только после Акта III.
- **Визуал** работает только если в репозитории есть `visual/QWEN_ADAPTER_v2.md`.
